# Orbit Workspace – AI Infinite Planner

A modern landing page concept built while learning frontend development with HTML, CSS, GitHub, and StackBlitz.

Orbit Workspace is a fictional AI productivity platform designed to transform messy thoughts, Slack messages, files, and planning chaos into a clean visual workspace.

## Technologies Used
- HTML
- CSS
- GitHub
- StackBlitz

## What I Practiced
- Building a modern landing page layout
- Creating a sticky navigation bar
- Using Flexbox for layout structure
- Working with CSS variables using `:root`
- Designing dark-mode UI aesthetics
- Creating glassmorphism-style cards
- Adding hover effects and transitions
- Improving visual hierarchy and spacing
- Structuring reusable CSS styles

## Project Goal
The goal of this project was to practice recreating the feel of a modern SaaS/productivity startup landing page while improving frontend layout and UI design skills.

## Features
- Sticky navigation bar
- Modern dark UI
- Gradient/glow styling
- CTA buttons with hover effects
- Floating dashboard cards
- Responsive-centered hero layout
